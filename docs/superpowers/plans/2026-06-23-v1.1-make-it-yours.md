# Anota v1.1 "Make It Yours" Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a selectable theme system (1 free + 5 Pro themes), a single optional "Anota Pro" in-app purchase that unlocks themes and theme-matched share cards, an image-based winner share card, and over-the-air updates, then ship it as v1.1.

**Architecture:** Pure-TS theme registry and entitlement logic (unit-tested, RN-free) sit under a React `ThemeProvider` that mirrors the existing `LanguageProvider`. Every component swaps its module-level `StyleSheet.create(colors...)` for a memoized `makeStyles(theme)` factory reading from `useTheme()`. Purchases use native billing (`react-native-iap`) with no third-party server. The share card is a hidden component captured to an image with `react-native-view-shot` and shared via `expo-sharing`.

**Tech Stack:** Expo SDK 54, React Native 0.81, React 19, TypeScript (strict), AsyncStorage, react-native-iap, react-native-view-shot, expo-sharing, expo-updates, Jest + ts-jest.

**Phasing:** Phases are ordered so the app is always working. After Phase 4 the theme system works end-to-end for free users (Pro themes visible but locked). Phase 5 makes Pro purchasable, Phase 6 adds the share card, Phase 7 ships. Each phase ends on a green typecheck + tests.

**Standing constraints:** No em dashes anywhere (code, comments, copy, commits). Free themes must look pixel-identical to live v1.0 (Classic Felt is the regression baseline). The app stays fully functional without buying anything.

```typescript
// src/theme/entitlement.ts
// Pure gating logic. No React Native imports.
import { getTheme, FREE_THEME_ID, Theme, ThemeId } from './themes';

export function isThemeLocked(theme: Theme, proUnlocked: boolean): boolean {
  return theme.tier === 'pro' && !proUnlocked;
}

export function resolveActiveThemeId(requestedId: string, proUnlocked: boolean): ThemeId {
  const theme = getTheme(requestedId);
  return isThemeLocked(theme, proUnlocked) ? FREE_THEME_ID : theme.id;
}
```

```typescript
// src/theme/ThemeProvider.tsx
import React, { createContext, useCallback, useContext, useEffect, useState } from 'react';
import AsyncStorage from '@react-native-async-storage/async-storage';
import { getTheme, Theme, ThemeId } from './themes';
import { resolveActiveThemeId } from './entitlement';

const STORAGE_KEY = '@anota:theme:v1';

interface Persisted {
  themeId: ThemeId;
  proUnlocked: boolean;
}

interface ThemeControls {
  themeId: ThemeId;
  setThemeId: (id: ThemeId) => void;
  proUnlocked: boolean;
  setProUnlocked: (v: boolean) => void;
  hydrated: boolean;
}

const ThemeValueContext = createContext<Theme | null>(null);
const ThemeControlsContext = createContext<ThemeControls | null>(null);

export function ThemeProvider({ children }: { children: React.ReactNode }) {
  const [themeId, setThemeIdState] = useState<ThemeId>('classic');
  const [proUnlocked, setProUnlockedState] = useState(false);
  const [hydrated, setHydrated] = useState(false);

  useEffect(() => {
    (async () => {
      try {
        const raw = await AsyncStorage.getItem(STORAGE_KEY);
        if (raw) {
          const p = JSON.parse(raw) as Partial<Persisted>;
          const pro = typeof p.proUnlocked === 'boolean' ? p.proUnlocked : false;
          const id = resolveActiveThemeId(String(p.themeId ?? 'classic'), pro);
          setProUnlockedState(pro);
          setThemeIdState(id);
        }
      } catch {
        // ignore, fall back to defaults
      }
      setHydrated(true);
    })();
  }, []);

  const persist = useCallback((next: Persisted) => {
    AsyncStorage.setItem(STORAGE_KEY, JSON.stringify(next)).catch(() => {});
  }, []);

  const setThemeId = useCallback(
    (id: ThemeId) => {
      const resolved = resolveActiveThemeId(id, proUnlocked);
      setThemeIdState(resolved);
      persist({ themeId: resolved, proUnlocked });
    },
    [proUnlocked, persist],
  );

  const setProUnlocked = useCallback(
    (v: boolean) => {
      setProUnlockedState(v);
      // Keep the current theme; if it was a downgrade it gets re-resolved on next select.
      persist({ themeId, proUnlocked: v });
    },
    [themeId, persist],
  );

  const theme = getTheme(themeId);

  return (
    <ThemeControlsContext.Provider
      value={{ themeId, setThemeId, proUnlocked, setProUnlocked, hydrated }}
    >
      <ThemeValueContext.Provider value={theme}>{children}</ThemeValueContext.Provider>
    </ThemeControlsContext.Provider>
  );
}

export function useTheme(): Theme {
  const ctx = useContext(ThemeValueContext);
  if (!ctx) throw new Error('useTheme must be used inside <ThemeProvider>');
  return ctx;
}

export function useThemeControls(): ThemeControls {
  const ctx = useContext(ThemeControlsContext);
  if (!ctx) throw new Error('useThemeControls must be used inside <ThemeProvider>');
  return ctx;
}
```

After EACH task: `npx tsc --noEmit` must be clean. Free-theme appearance must be unchanged (Classic Felt maps 1:1 to the old `colors`).

```typescript
// src/iap/purchases.ts
import {
  initConnection,
  endConnection,
  getProducts,
  requestPurchase,
  getAvailablePurchases,
  finishTransaction,
  type Purchase,
} from 'react-native-iap';

export const PRO_PRODUCT_ID = 'dev.anota.pro';

export async function initIap(): Promise<void> {
  await initConnection();
}

export async function endIap(): Promise<void> {
  try {
    await endConnection();
  } catch {
    // ignore
  }
}

export async function getProPriceLabel(): Promise<string | null> {
  try {
    const products = await getProducts({ skus: [PRO_PRODUCT_ID] });
    return products[0]?.localizedPrice ?? null;
  } catch {
    return null;
  }
}

function ownsPro(purchases: Purchase[]): boolean {
  return purchases.some((p) => p.productId === PRO_PRODUCT_ID);
}

export async function restorePro(): Promise<boolean> {
  try {
    const purchases = await getAvailablePurchases();
    return ownsPro(purchases);
  } catch {
    return false;
  }
}

export async function buyPro(): Promise<boolean> {
  const purchase = await requestPurchase({ sku: PRO_PRODUCT_ID });
  const result = Array.isArray(purchase) ? purchase[0] : purchase;
  if (result) {
    await finishTransaction({ purchase: result, isConsumable: false });
    return true;
  }
  return false;
}
```
(If the installed react-native-iap version's signatures differ, adapt the call shapes; the contract this module exposes must stay: `initIap`, `endIap`, `getProPriceLabel`, `restorePro`, `buyPro`.)
